# model/

This folder contains trained model artifacts generated by `train.py`.

| File | Description |
|---|---|
| `classifier.pkl` | Trained OneVsRest Logistic Regression (~88MB) |
| `tfidf.pkl` | Fitted TF-IDF vectorizer (75k features) |
| `mlb.pkl` | MultiLabelBinarizer (116 labels) |
| `thresholds.pkl` | Per-label tuned thresholds |

> These files are not included in the repo due to size.
> Run `python train.py` from the root directory to regenerate them.
